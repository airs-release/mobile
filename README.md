# AIRS Mobile

AIRS 앱의 Android/iOS 개발 및 UI 레퍼런스 공유용 저장소입니다.

## UI Reference

GitHub Pages 배포 대상은 `/docs` 디렉토리입니다.

배포 후 팀원 공유 링크 예시:

```text
https://<github-id-or-org>.github.io/mobile/
```

## Repository Layout

```text
mobile/
  android/   # Android 앱 코드 위치
  ios/       # iOS 앱 코드 위치
  docs/      # GitHub Pages 정적 사이트
```

## GitHub Pages 설정

GitHub 저장소에서 아래처럼 설정합니다.

```text
Settings -> Pages
Source: Deploy from a branch
Branch: main
Folder: /docs
```
