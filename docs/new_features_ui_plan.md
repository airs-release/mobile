# AIRS 신규 기능 회의 메모 정리

작성일: 2026-05-24

## 1. 기존 기능

- LCD 패널로 현재 온습도 표시
- 사각형 디자인의 데스크 거치형 기기

## 2. 차별점 및 신규 기능 아이디어

### 2.1 적외선 센서를 활용한 에어컨 제어

- 적외선 센서를 통해 에어컨 제어 기능을 추가한다.
- 신규 기능의 핵심 동작부로 본다.

### 2.2 20분 후 온습도 예측 및 LCD 표시

- 현재 온습도뿐 아니라 향후 20분 후 온습도를 예측해 LCD 패널에 표시한다.
- 예시:
  - 현재 에어컨이 켜져 있음
  - 현재 온도는 23도
  - 외부 온도는 30도
  - 20분 후 26도까지 상승할 것으로 예측
  - 사용자는 너무 덥다는 의미로 보고 에어컨을 미리 켤지 판단할 수 있음
- 기존 프로젝트의 주요 기능인 향후 20~30분 후 온도 예측 기능은 유지한다.

### 2.3 외부 온도 대비 온도 차이 표시

- 바깥 날씨와 방 안 온도를 비교할 수 있게 한다.
- 목적:
  - 바깥 날씨가 실내보다 얼마나 더운지 비교 가능
  - 사용자 입장에서 재미와 실용성을 동시에 줄 수 있음
- 기상청 API 활용을 검토한다.
- 기상청 측정 기준 위치는 Wi-Fi 또는 앱 설정으로 지정한다.
- 습도 비교는 사용자 관심도가 낮을 것으로 예상되어 우선순위에서 제외한다.

### 2.4 사람 부재 시 방 온도 유지 기능

- 방에 사람이 없을 때 방 온도 유지 기능을 활성화할지 선택할 수 있게 한다.
- 목적:
  - 방이 너무 더워지면 온도에 영향을 받는 물품이 손상될 수 있음
  - 예: 식품, 화장품, 전자기기 등
  - 사용자가 없을 때도 일정 온도 이상으로 올라가지 않도록 보호
- 인체 감지에는 mmWave 센서 활용을 검토한다.
- 앱 알림으로 방 온도가 너무 더운지 안내한다.
- 예시 동작:
  - 방 온도가 28도 이상이면 사용자에게 에어컨 켜기 여부를 묻는다.
  - 이후 일정 온도 이하, 예: 26도 이하로 내려가면 에어컨을 자동으로 끈다.
  - 도중에 사람이 감지되면 자동 끄기 동작을 비활성화한다.

### 2.5 사용자 GPS 기반 자동 에어컨 활성화

- 사용자의 앱 GPS를 파악해 일정 거리 안에 들어오면 자동으로 에어컨을 활성화한다.
- 목적:
  - 더운 날씨에 집에 들어왔을 때 바로 시원한 환경을 제공
  - 학교나 강의실 등 더운 외부 환경에서 돌아오는 사용자의 체감 만족도를 높일 수 있음
- 예시 동작:
  - 퇴근 후 집으로 올 때 사용자가 1km 이내에 들어오면 미리 작동
  - 자동차로 이동하는 경우 1km, 도보 이동의 경우 500m 이내 감지를 고려
  - 설정으로 매번 묻지 않고 항상 자동으로 켜는 기능 추가
- 반대 상황도 고려한다.
  - 에어컨을 켠 채 외출한 경우 앱 GPS로 500m 밖임을 감지
  - 또는 10~20분 이상 사람이 감지되지 않으면 부재 상태로 간주
  - 사용자에게 에어컨 끄기 여부를 물어본다.
  - 전기세 낭비 방지 기능으로 볼 수 있다.

## 3. 아이디어 장점

- 기존 타깃인 대형 강의실 중심에서 가정용 사용성으로 확장할 수 있다.
- 가정용 온습도계 기능 확장으로 시장성을 강화할 수 있다.
- 편의성 대폭 추가로 사용자 체감성이 높아질 것으로 예상된다.
- 기존 프로젝트의 주요 기능인 향후 20~30분 후 온도 예측 기능을 버리지 않고 활용할 수 있다.

## 4. 아쉬운 점 및 검토 필요 사항

- 프로젝트에 대한 학회원들의 관심도가 낮을 것으로 예상된다.
- 시장성 확보가 불확실하며, 이미 존재하는 유사 기술이 있을 가능성이 있다.
- 시장 조사와 차별성 검토가 필요하다.

## 5. UI로 풀어야 할 대표 화면

### 5.1 예측 대시보드

- 현재 실내 온도와 습도
- 20분 후 예측 온도
- 외부 온도
- 현재 에어컨 상태
- LCD 패널에 표시될 정보 미리보기
- 사용자가 미리 냉방할지 판단할 수 있는 CTA

### 5.2 외부 온도 비교 화면

- 현재 방 온도와 외부 온도 비교
- 외부 온도 기준 위치 설정
- 기상청 API 기반 데이터 표시
- 온도 차이를 직관적으로 보여주는 비교 카드 또는 게이지

### 5.3 부재중 방 온도 보호 설정

- 사람이 없을 때 방 온도 유지 기능 on/off
- 보호 기준 온도 설정
- mmWave 인체 감지 상태
- 자동 켜기/끄기 조건
- 알림 설정

### 5.4 GPS 기반 자동 활성화 설정

- 집 주변 진입 거리 설정
- 자동차/도보 기준 거리 선택
- 자동 에어컨 켜기 on/off
- 외출 감지 시 끄기 알림
- 사람 감지와 GPS 감지의 조합 상태
